Cours en ligne et MOOC

  • Initiation au Machine Learning. Cours interne de Google rendu public en 2018. Décrit très bien les fondamentaux. Disponible en français et en anglais. Utilise TensorFlow.

  • Coursera : Machine Learning. Une référence depuis plusieurs années. Enseigné par Andrew Ng. Utilise Matlab/Octave.

  • Practical Deep Learning For Coders. Une approche pragmatique et “top-down” : partir d’applications concrètes pour aller vers les concepts théoriques. Utilise une librairie basée sur PyTorch.

  • Coursera : Deep Learning Specialization. Une série de cinq cours enseignés par Andrew Ng. Utilise Python et TensorFlow.

  • EPFL Deep Learning Course. Un cours très complet de l’Ecole Polytechnique Fédérale de Lausanne. Utilise PyTorch.

Livres

  • Python Data Science Handbook de Jake VanderPlas. Présente en détail les outils fondamentaux : NumPy, Pandas, Matplotlib et Scikit-Learn.

  • Hands-On Machine Learning with Scikit-Learn & TensorFlow d’Aurélien Géron. Une approche concrète qui couvre de nombreux algorithmes de ML.

  • ($) Deep Learning with Python de François Chollet. Une introduction claire au DL avec Keras, écrite par son créateur.

  • Neural networks and deep learning de Michael Nielsen. Une étude détaillée du fonctionnement des réseaux de neurones. Utilise Python.
